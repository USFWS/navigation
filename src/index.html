Responsive Navigation

Features

  • Off-canvas navigation menu saves screen real estate
  • Supports unlimited sub-menus
  • Automatically injects list items for navigating between submenus
  • Support for keyboard interaction
    • When the menu is open press escape to close the menu
    • Automatically manages tabindex which prevents you from tabbing into a hidden menu
