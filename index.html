Responsive Navigation

Features

  • Mark up your navigation with standard unordered lists
  • Supports unlimited sub-menus
  • Off-canvas navigation menu saves screen real estate
  • Configurable to appear on left or right side on larger screens
  • Automatically resets to initial state when closed
  • Automatically injects list items for navigating between submenus
  • Support for keyboard interaction
    • When the menu is open press escape to close the menu
    • Up key: focus the last tabbable element
    • Down key: focus the nextx tabbable element
    • Left key: Close the current submenu, or the whole menu if you're on the root ul element
    • Right key: If the currently focused element contains a submenu, activate that submenu
    • Automatically manages tabindex which prevents you from tabbing into a hidden menu
